# RSS Reader

Сервис для агрегации RSS-потоков - специализированный формат, предназначенный для описания лент новостей, анонсов статей и других материалов. 

[Посетить сайт](https://frontend-project-11-jet-psi.vercel.app/)

`node v22.9.0`
`npm 10.9.0`

### Установка и запуск

`git clone`
`npm ci`
`npm run build`
